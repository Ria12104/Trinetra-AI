# Trinetra AI

Trinetra AI is a beginner-friendly cybersecurity education web app that explains cyber attacks and online threats in simple language using Google Gemini AI.

Inspired by cyber defense, awareness, and futuristic terminal aesthetics, the project helps users understand cybersecurity concepts through clear and practical explanations.

## Features

- Explains cybersecurity attacks in simple English
- AI-powered responses using Google Gemini API
- Beginner-friendly learning experience
- Matrix-inspired UI and styling
- Strict cybersecurity-only responses
- Fast and lightweight frontend project

## Technologies Used

- HTML
- CSS
- JavaScript
- Google Gemini API
- Vercel (deployment)

## Example Topics

- Phishing
- Ransomware
- SQL Injection
- Social Engineering
- Malware
- DDoS Attacks
